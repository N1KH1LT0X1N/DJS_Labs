# DJS_Labs - Comprehensive Computer Science Learning Repository

A comprehensive repository containing multiple laboratory exercises and projects across key computer science domains: Corporate Finance & Project Management (CFPM), Data Structures & Algorithms (DSA), Machine Learning (ML), Spatial Data Science (SDS), and Web Development (Web).

## 📁 Repository Structure

### **CFPM** - Corporate Finance & Project Management
Laboratory exercises focusing on financial concepts, time value of money, loan amortization, and financial analysis using Python.
- **Lab 0**: Time Value of Money calculations
- **Lab 1**: Loan Amortization Schedules
- **Lab 2-5**: Advanced financial analysis and project management

### **DSA** - Data Structures & Algorithms
Foundational CS courses implemented in C, covering core data structures and algorithms.
- **Lab 0-1**: Basic C programming and sorting
- **Lab 2-5**: Arrays, Stacks, and Queue implementations
- **Lab 6**: Searching and Sorting algorithms
- **Lab 7**: Hashing techniques
- **Lab 8**: Linked Lists
- **Lab 9**: Stack implementations
- **Lab 10**: Circular Queue

### **ML** - Machine Learning
Hands-on machine learning projects using Python (scikit-learn, pandas, NumPy). Includes data preprocessing, model training, and deployment.
- **Lab 0**: Complete ML pipeline with data cleaning and model deployment
- **Lab 1**: Regression analysis on Boston Housing dataset
- **Lab 2**: Multi-dataset analysis (California Housing, Iris, Wine)
- **Lab 3**: Classification and clustering on medical datasets
- **Lab 4**: Advanced ML concepts
- **Lab 5**: Specialized ML techniques

### **SDS** - Spatial Data Science
Geospatial analysis, data visualization, and spatial statistical methods.
- Multiple laboratory exercises covering spatial analysis techniques

### **Web** - Web Development
Full-stack web development projects and exercises.
- Comprehensive web development labs and projects

## 🚀 Setup & Installation

```bash
# Clone the repository
git clone <repository-url>
cd DJS_Labs

# Create a virtual environment (optional but recommended for Python labs)
python -m venv .venv

# Activate virtual environment
# On Windows:
.venv\Scripts\activate
# On macOS/Linux:
source .venv/bin/activate

# Install Python dependencies
pip install -r requirements.txt
```
