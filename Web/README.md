## 📞 Troubleshooting

### Problem: CSS not loading
**Solution:** Make sure CSS file (1_external_styles.css) is in same lab2 folder as HTML file

### Problem: Forms not saving data
**Solution:** Check browser console (F12) for errors, ensure JavaScript is enabled

### Problem: Mobile view not responsive
**Solution:** Hard refresh (Ctrl+F5), use DevTools device toggle (Ctrl+Shift+M)

### Problem: Bootstrap styling not showing
**Solution:** Check internet connection (Bootstrap loaded from CDN), try different browser
